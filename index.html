How Option B is calculated:
Cases per Position = Cases per Layer x Layers that fit within (Rack Height - Pallet Height)
Actual 60" Positions = Boxes / Cases per Position
72" Billable Positions = Actual 60" Positions x (Rack Height / Billable Height)
Monthly Charge = 72" Billable Positions x Customer Rate
This is an estimate tool. It assumes regular stacking and that the best carton orientation shown is operationally acceptable. If a carton is crush-sensitive, label-sensitive, or must stay upright, update the row note and override the dimensions manually if needed.
